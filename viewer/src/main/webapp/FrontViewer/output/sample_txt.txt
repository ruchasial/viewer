2) INSTRUCTIONS D'INSTALLATION
_____________________________

Étapes :
	
1.  (Version disque uniquement) Insérez le disque 1 de Grand Theft Auto V dans votre lecteur de DVD-ROM. Si le programme d'installation ne se lance pas, ouvrez le répertoire de votre lecteur de DVD-ROM et cliquez sur l'icône "Setup".

2.  L'installation de Grand Theft Auto V peut alors commencer. Suivez les instructions à l'écran jusqu'à la fin de l'installation.

3.  D'autres composants tels que DirectX et les logiciels Microsoft .NET et Visual C++ seront ensuite installés si nécessaire. Veuillez suivre les instructions à l'écran.

4.  Pour jouer à Grand Theft Auto V, vous n'avez pas besoin du DVD dans votre lecteur.

5.  Cliquez sur l'icône "Grand Theft Auto V" pour lancer Grand Theft Auto V. Le jeu téléchargera et installera d'éventuelles mises à jour.

6.  Vous devrez créer un compte Social Club ou vous connecter à un compte existant.

7.  Vous devrez ensuite saisir votre code d'activation Rockstar pour activer le produit (attention, une seule activation autorisée !) via Internet. Vous trouverez le code d'activation Rockstar au dos du manuel ou dans l'email que vous recevez au moment de l'achat du jeu.

Sachez que le jeu inclut une vérification de la date de sortie et ne pourra pas être activé avant la sortie officielle du produit. Si vous n'arrivez pas à vous connecter ou si la procédure échoue, rendez-vous sur www.rockstargames.com/support pour plus d'informations. Veuillez conserver votre manuel de jeu dans un endroit sûr car aucun code d'activation Rockstar ne pourra être réédité.

6) INFORMATIONS SUR LES CONTRÔLES DE LA VERSION PC
_____________________________

Grand Theft Auto V sur PC est compatible avec la souris et le clavier, et les périphériques Xinput comme la manette Xbox 360 pour Windows.

Tous les types de périphériques sont interchangeables. Par exemple, vous pouvez utiliser la souris et le clavier pour les commandes à pied, puis passer à la manette Xbox 360 à tout moment sans avoir à changer vos options de jeu.

Pour la souris, le clavier et les manettes Xinput, l'aide en jeu se mettra automatiquement à jour pour afficher les commandes correspondant au dernier périphérique de jeu utilisé.

7) CONTACT ET ASSISTANCE
___________________________________

Si vous rencontrez des problèmes techniques, rendez-vous sur www.rockstargames.com/support pour obtenir les informations de contact les plus à jour et les réponses aux questions fréquemment posées. Nous proposons une assistance via Internet, email, téléphone et Twitter.

Remarque : si vous avez besoin d'aide à propos d'un titre PC, veuillez joindre vos fichiers DxDiag et MSinfo à votre message lorsque vous nous contactez. 

Pour savoir comment obtenir ces fichiers, veuillez procéder comme suit :

Exporter un rapport DxDiag :
	- Dans le menu Démarrer, tapez "dxdiag" (sans les guillemets) dans le champ Rechercher et appuyez sur Entrée
	- Attendez que la barre de progression verte soit remplie puis appuyez sur le bouton "Enregistrer toutes les informations"
	- Nommez le fichier avec des informations pertinentes, comme votre nom et la date, et sauvegardez-le dans un emplacement facile d'accès, sur le bureau Windows par exemple, en appuyant sur le bouton "Enregistrer"
	- Appuyez sur le bouton "Quitter" pour fermer le programme


Exporter un rapport MSInfo :
	- Dans le menu Démarrer, tapez "msinfo32" (sans les guillemets) dans le champ Rechercher et appuyez sur Entrée
	- Lorsque la fenêtre "Informations système" est ouverte, cliquez sur "Fichier" dans la barre d'outils puis cliquez sur "Exporter..."
    	- Nommez le fichier avec des informations pertinentes, comme votre nom et la date, et sauvegardez-le dans un emplacement facile d'accès, sur le bureau Windows par 	exemple, en appuyant sur le bouton "Enregistrer"
    	- Le processus d'export peut prendre quelques instants, attendez que la barre de progression soit remplie, ensuite vous pouvez fermer ce programme


Veillez à inclure une description détaillée du problème rencontré ainsi que le texte des éventuels messages d'erreur que vous recevez. Si vous pouvez également joindre une capture d'écran du message d'erreur en plus des deux fichiers textes, cela nous sera d'une grande aide.


Faire une capture d'écran d'un message d'erreur qui apparaît :
	- En vous assurant de ne fermer aucune fenêtre contenant un message d'erreur, cliquez sur la barre de titre pour que cette fenêtre devienne la fenêtre active
    	- Appuyez sur les touches "Alt" et "Impr écran" en même temps
    	- Assurez-vous de laisser la fenêtre ouverte, puis dans le menu Démarrer, tapez "mspaint" (sans les guillemets) dans le champ Rechercher et appuyez sur Entrée
    	- Dans Paint, appuyez sur les touches CTRL et V en même temps pour coller la capture d'écran dans le programme. Vous pouvez également aller dans le menu Édition 	sous la barre de titre et sélectionner "Coller".
    	- Vous devriez voir la fenêtre du message d'erreur et uniquement celle-ci collée dans le programme. Si vous voyez tout le bureau, retournez aux étapes précédentes 	et assurez-vous de presser la touche "Alt" en même temps que la touche "Impr écran"
   	- Cliquez sur "Fichier" dans la barre d'outils, puis cliquez sur "Enregistrer sous"
    	- Dans la liste déroulante "Type" en bas de la fenêtre, sélectionnez "JPEG" pour réduire la taille du fichier avant de l'envoyer à l'assistance
    	- Nommez le fichier avec des informations pertinentes, comme votre nom et la date, et sauvegardez-le dans un emplacement facile d'accès, sur le bureau Windows par 	exemple, en appuyant sur le bouton "Enregistrer"
   	- Fermez le programme Paint une fois la sauvegarde effectuée
